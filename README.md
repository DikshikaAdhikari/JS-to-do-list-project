# 📝 JavaScript To-Do List

A modern and interactive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**. This project allows users to efficiently organize daily tasks by adding, completing, editing, and deleting tasks through a clean and responsive interface.

## 🌐 Live Demo

🔗 **Live Website:** https://dikshikaadhikari.github.io/JS-to-do-list-project/

## 📌 Project Overview

This project demonstrates the fundamentals of JavaScript by creating a fully functional task management application. It focuses on DOM manipulation, event handling, dynamic content creation, and user interaction while providing a smooth and intuitive user experience.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ✏️ Edit existing tasks
- 🗑️ Delete individual tasks
- 🧹 Clear completed tasks
- 📊 Display task statistics
- 🎨 Attractive and responsive user interface
- ⚡ Instant updates without refreshing the page
- 📱 Mobile-friendly design

## 🛠️ Technologies Used
